Frequently asked questions

Does resizing an image in inches cost anything?
Nothing at all — there is no account to set up, no watermark on the output, and no limit to how many images you can put through it.
Is my image kept private?
Absolutely. HTML5 Canvas performs the resize right in your browser, which keeps the image off any server and on your device the whole way through.
Can I use it on a phone?
You can. Fire up YESPYQ in Chrome or Safari on an iPhone or Android, drop in the image, and save the result — nothing needs installing.
Which image formats can I upload?
Hand it JPG, JPEG, PNG, WEBP or HEIC/HEIF (where your browser can read them). It always writes a JPEG, which works cleanly with Indian government exam and ID portals.
Is any software installation required?
No. The tool lives entirely in the browser you already have — Chrome, Firefox, Edge or Safari — whether on a computer or a phone.
Which DPI is right for printing?
Aim for 300 DPI on typical prints like ID cards, exam photos and documents: 1 inch = 300 px, 1 cm ≈ 118 px, 1 mm ≈ 11.8 px. Because the tool counts in exact pixels, enter the pixel figures those conversions produce.
Why does it save as JPEG rather than PNG?
JPEG is the format nearly all Indian government exam and ID portals expect, and it stays much lighter than PNG while keeping the picture sharp — ideal for uploading through online forms.
