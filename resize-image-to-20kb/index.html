Why resize an image to 20 KB?

Most exam portals — UPSC, SSC, IBPS, SBI, NTA NEET and the rest — will only take images that sit inside a fixed KB range, and 20 KB is among the ceilings they ask for most often. Portals impose these caps because a heavier file breaches their limit and the upload form turns it away, which means you have to bring the size down before the application will register.

Doing this the traditional way — launching Paint, Photoshop, or GIMP — pulls in installed software and a fiddly sequence of steps. Because this resizer runs inside the browser instead, it settles the whole thing in one click, with no upload, no account, and no fee.

Tips for the best result

  • Feed it the crispest source file you can find — the smarter the starting detail, the more the compressor preserves as it drives toward the target.
  • When the image is an exam photo, pick a well-lit, in-focus shot set against a plain white or pale backdrop.
  • Trim the frame close to the subject, leaving only a slim margin, so a larger share of the pixels survives once the file is shrunk.
  • Should the result come out soft, go back to a bigger original and run it through once more.
Good to know: every file comes out as JPEG, the exact format Indian exam and ID portals are built to accept, so the upload goes through cleanly on the first attempt.

You can feed it JPG, JPEG, PNG, WEBP, and HEIC/HEIF where the browser allows. It writes the result as JPEG, the format Indian exam and ID portals reliably accept.

The tool holds on to as much clarity as the 20 KB budget permits. Squeezing down to a ceiling this small does soften the picture a touch, yet the finished file still clears exam-portal checks.
Why do exam portals set a KB file-size limit?
Sites such as SSC, UPSC, IBPS, NTA, and RRB put KB ceilings in place so their storage stays lean and uploads still complete quickly over weak connections. Cross the limit and the form simply refuses the file.
