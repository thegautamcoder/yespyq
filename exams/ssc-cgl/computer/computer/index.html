Q3
Which of the following is an application protocol?
aHTTP
bTCP
cIP
dUDP
✓ Correct answer: a) HTTP
ExplanationThe Hypertext Transfer Protocol (HTTP) is an application protocol used for distributed, collaborative, and hypermedia information systems, serving as the foundation of data communication for the World Wide Web.HTTP (Hypertext Transfer Protocol) – Correct.It operates at the Application Layer (Layer 7 of the OSI model) and is used to load webpages and transfer hypermedia documents.TCP (Transmission Control Protocol) – Incorrect.It is a Transport Layer protocol responsible for reliable, ordered, and error-checked delivery of data.IP (Internet Protocol) – Incorrect.It is a Network Layer (Internet Layer) protocol responsible for addressing and routing packets across the internet.UDP (User Datagram Protocol) – Incorrect.It is a Transport Layer protocol used for fast, connectionless, and lightweight communication, often for streaming or DNS.Hence, the correct answer is "HTTP".
