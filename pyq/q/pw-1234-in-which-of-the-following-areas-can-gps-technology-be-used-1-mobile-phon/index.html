In which of the following areas can GPS technology be used?1. Mobile phone operations2. Banking operations3. Controlling the power gridsSelect the correct answer using the code given below:

a1 only
b2 and 3 only
c1 and 3 only
d1, 2 and 3
✓ Correct answer: d) 1, 2 and 3
ExplanationGPS technology uses a network of satellites to pinpoint a user’s location on Earth.By receiving signals from multiple satellites, GPS devices can calculate precise latitude, longitude, and altitude coordinates.This technology has revolutionized navigation, transportation, and many other fields.Options 1 is correct: Smartphones use GPS to determine the user’s precise location which enables location-based services such as navigation, ride-sharing apps, geotagging, and emergency location tracking.Option 2 is correct: Financial institutions use GPS-derived time stamps to record the exact time of transactions, aiding in audit trails and fraud detection.GPS ensures that banking systems across different locations operate in sync, which is crucial for real-time processing and settlements.Options 3 is correct: GPS provides precise time references that are vital for synchronizing the operations of power grids, helping in load balancing and efficient energy distribution.78
