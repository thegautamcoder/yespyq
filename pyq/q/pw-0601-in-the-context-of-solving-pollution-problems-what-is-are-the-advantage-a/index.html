In the context of solving pollution problems, what is/are the advantage/advantages of bioremediation techniques?1. It is a technique for cleaning up pollution by enhancing the same biodegradation process that occurs in nature.2. Any contaminant with heavy metals such as cadmium and lead can be readily and completely treated by bioremediation using microorganisms.3. Genetic engineering can be used to create microorganisms specifically designed for bioremediation.Select the correct answer using the code given below:

a1 only
b2 and 3 only
c1 and 3 only
d1, 2 and 3
✓ Correct answer: c) 1 and 3 only
