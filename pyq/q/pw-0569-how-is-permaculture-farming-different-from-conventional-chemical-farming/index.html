Permaculture farming discourages monocultural practices but in conventional chemical farming, monoculture practices are predominant.2. Conventional chemical farming can cause an increase in soil salinity but the occurrence of such a phenomenon is not observed in permaculture farming.3. Conventional chemical farming is easily possible in semi-arid regions but permaculture farming is not so easily possible in such regions.4. Practice of mulching is very important in permaculture farming but not necessarily so in conventional chemical farming.Select the correct answer using the code given below

a1 and 3

c4 only
d2 and 3
