In the cities of our country, which among the following atmospheric gases are normally considered in calculating the value of the Air Quality Index?1. Carbon dioxide2. Carbon monoxide3. Nitrogen dioxide4. Sulfur dioxide5. MethaneSelect the correct answer using the code given below.

a1, 2 and 3 only
b2, 3 and 4 only
c1, 4 and 5 only
