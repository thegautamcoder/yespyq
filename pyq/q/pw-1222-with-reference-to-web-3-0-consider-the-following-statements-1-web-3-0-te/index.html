With reference to Web 3.0, consider the following statements:1. Web 3.0 technology enables people to control their own data.2. In Web 3.0 world, there can be blockchain based social networks.3. Web 3.0 is operated by users collectively rather than a corporation.Which of the statements given above are correct?

a1 and 2 only
b2 and 3 only
c1 and 3 only
d1, 2 and 3
✓ Correct answer: d) 1, 2 and 3
ExplanationStatement 1 is correct: A fundamental principle of Web 3.0 is to grant users ownership and control over their personal data, reducing reliance on centralized entities.This shift aims to enhance privacy and security for individuals.Statement 2 is correct: Web 3.0 leverages blockchain technology to create decentralized applications, including social networks.These platforms operate without central authorities, offering users greater control over content and data.Statement 3 is correct: Decentralization is a hallmark of Web 3.0, where control is distributed among users instead of being held by centralized corporations.This collective operation is facilitated through decentralized networks and consensus mechanisms.Feature Web 1.0 Web 2.0 Web 3.0 Web 4.0 Timeframe Roughly 1989- 2005 Roughly 2005-present Evolving Emerging Key Characteristic "Read-only" web "Interactive" or "social" web "Semantic" or "decentralized" web "Intelligent" or "symbiotic" web Focus Information consumption User- generated content, social interaction Data understanding, decentralization Intelligent connectivit
